# PLAY3 Technical Analysis

This directory contains detailed technical documentation
about the reconstruction of PLAY3 for the SHARP PC-E500.

## Documents

- play3_algorithm.md  
  Overview of the PLAY3 playback algorithm.

- play3_mixing.md  
  Explanation of the three-voice software mixer.

- play3_timing.md  
  Timing analysis of the sound generation loop.

- play3_memory_map.md  
  RAM structure and internal variables.

- play3_binary_analysis.md  
  Binary disassembly and comparison notes.

- PLAY3_algorithm_reconstruction_note.md  
  Reconstruction notes from the original magazine source.

- PLAY3_binary_comparison_summary.txt  
  Binary comparison results between reconstructed builds.
