# PLAY3 reconstruction process

The original PLAY3 program was published in:

Pocket Computer Journal
November 1993 issue

The reconstruction process used in this repository:

1. Magazine scan preservation
2. OCR transcription of the assembly listing
3. Manual correction of OCR errors
4. Reconstruction into XASM-compatible source
5. Commented source versions (Japanese / English)

Files in this repository:

play3_scaned.asm
    OCR transcription of the magazine listing

play3_pushu.asm
    corrected reconstruction source

play3_jp.asm
    Japanese commented version

play3_en.asm
    English commented version

The assembler code is identical between all versions.
Only comments differ.
