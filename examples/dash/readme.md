# Dash! – Example Music for PLAY3

Example RPG floor BGM written for the PLAY3 buzzer music routine
for the SHARP PC-E500 pocket computer.

## Files

- dash.bas – early prototype version
- dash1.bas – intermediate version
- dash2.bas – final version

## Original timestamps

These files retain their original timestamps from 1994:

- dash.bas – 1994-05-11
- dash1.bas  – 1994-05-11
- dash2.bas – 1994-05-11

## Background

The music "Dash!" was composed around 1994 for a personal RPG project.

It demonstrates three-voice polyphonic music generated from the
internal buzzer using the PLAY3 software mixing routine.
