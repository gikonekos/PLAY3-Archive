# Bottakuri Shouten – Early Music Program

Music program associated with the piece **"Bottakuri Shouten" (ボッタクリショウテン)**  
composed by **Kenkichi Motoi** on **1994-05-11**.

The program is written for the **PLAY2L buzzer music routine** for the  
**SHARP PC-E500 pocket computer series**.

---

## Files

- `BOTTA.BAS` – original BASIC music program OLD version
- `BOTTA2.BAS` – original BASIC music program

---

## Description

`BOTTA.BAS` contains an early version of the music piece  
**"Bottakuri Shouten"**, written in 1994.

The program uses the **PLAY2L** music routine to generate multi-voice music  
from the internal piezo buzzer of the SHARP PC-E500 series.

The music structure follows a typical game BGM format with sections  
(A / B / C / D) and looping playback.

---

## Historical context

The piece **"Bottakuri Shouten"** was later used as background music in the  
PC-9801 MS-DOS game:

**Space Panicco (すぺーすぱにっ娘)**

Release information:

- Release date: **1994-09-20**
- Program: **Geimu Shokunin**
- Music & graphics: **Kenkichi Motoi**

This BASIC program therefore preserves an **early form of a piece later used
in a PC-98 game soundtrack**.

---

## Related music driver

The program uses the **PLAY2L** buzzer music routine written by:

**Ryu (Tatsuya Kobayashi / 小林龍也)**

Related driver lineage:

PLAY2 → PLAY2L → PLAY3

PLAY3 was later published in *Pocket Computer Journal* (1993).

---

## Notes

The title line in the BASIC source appears garbled in UTF-8 due to character
encoding differences.

**「ボッタクリショウテン」**

220 PLAY "+C0R2C0R2E0R2G0R2::E0R2R0R2R0R2R0R2
230 *B :PRINT "B";
240 PLAY "+E2R0G0R2C0R2G0R2::O1G2

260 PLAY "D0R2A0R2D0R2A0R2

300 PLAY "C0R2G0R2C0R2G0R2
310 PLAY "+C0R2G0R2C0R2G0R2
320 A=A+1:IF A<4 GOTO *B
330 *C:PRINT "C";
